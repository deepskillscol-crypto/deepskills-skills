## 🎯 ¿Cómo usar estas skills?

### Opción A — Claude.ai (Plan Free, Pro, Max)
1. Ve a **Settings → Capabilities** → activa *Code execution and file creation*
2. Ve a **Customize → Skills**
3. Descarga el archivo `SKILL.md` del rol que necesitas
4. Súbelo como skill personal

### Opción B — Plataforma Deepskills (próximamente)
Accede directamente desde tu cuenta en la plataforma MJL — las skills se cargan automáticamente según tu perfil de rol.
